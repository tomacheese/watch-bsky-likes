# AGENTS.md

## 目的
このプロジェクトにおける AI エージェント共通の作業方針を定義します。

## 基本方針
- **会話言語**: 日本語で行う。
- **コード内コメント**: 日本語で記載する。
- **エラーメッセージ**: 英語で記載する。
- **コミットメッセージ**: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) に従い、`<description>` は日本語で記載する。
- **スタイル**: 日本語と英数字の間に半角スペースを入れる。

## 判断記録のルール
エージェントが設計上の決定や重要な変更を行った場合、以下を明示してください。
- 決定した内容とその理由
- 検討した代替案
- 前提条件、仮定、および不確実性

## 開発手順
1. **理解**: `package.json` やソースコードを読み、プロジェクトの構造を理解する。
2. **準備**: `pnpm install` で依存関係をセットアップする。
3. **実装**: 既存のコーディングスタイルや規約に従い、機能を実装・修正する。
4. **検証**: `pnpm lint` や動作確認を行い、品質を担保する。

## セキュリティ / 機密情報
- `data/config.json` などの設定ファイルに含まれる API キーやトークンを Git にコミットしない。
- ログに機密情報を出力しない。

## リポジトリ固有の注意事項
- Bluesky API の仕様変更に注意する。
- Discord Webhook/Bot の設定が正しいか確認する。
- `data/` ディレクトリ内のファイルの読み書き権限に注意する。
